Що таке HTML?

HTML (HyperText Markup Language) — це стандартна мова розмітки для створення веб-сторінок. Вона структурує контент за допомогою елементів: заголовків, абзаців, посилань, зображень тощо.

HTML працює разом з CSS та JavaScript для побудови сучасних сайтів.
